To Market

To Market is a web application that helps visitors and locals easily find a farmers market near them. Users search by zip code and see farmers market results plotted on a map as well as in a list. I built this for a 1 week sprint assignment during General Assembly's Web Development Immersive Program.

Technologies Used: Javascript, Node.js, Express, MongoDB, Mongoose, jQuery, AJAX, Mapbox API, USDA National Farmers Market Directory API, Google Fonts API.

Canopy (Group Project)

Canopy is a go-to sharing resource for web developers, programmers and engineers. We set out to build a resource where live projects, code snippets, and interaction between developers are all in one place. Live projects are embedded and rendered on the site so that visitors can interact with them without leaving Canopy.

Technologies Used: Ruby on Rails, postgreSQL, AJAX, jQuery, Screenshot Machine API, highlight.js, Paperclip.

Simplfy Local

Mastercard Masters Of Code Hackathon | Top 6 Semi-Finalist

In 24 hours, my team and I built a mobile-friendly web app where users can find and shop at locally owned businesses near them. I lead the front-end development by designing layouts, integrating maps, and developing the pitch to the judges.

Technologies Used: Ruby on Rails, login and authentication with Twitter Digits API, mobile payments with the MasterCard Simplify Commerce API, HTML 5 location services, Mapbox.
