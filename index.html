Projects      About Me







To Market

To Market is a web application that helps visitors and locals easily find a farmers market near them. Users search by zip code and see farmers market results plotted on a map as well as in a list. I first built To Market during a 1-week sprint assignment at General Assembly and am working on a second iteration in MEAN stack where I'm building my own API using the USDA data.

Technologies Used: Javascript, Node.js, Express, MongoDB, Mongoose, jQuery, AJAX, Mapbox API, USDA National Farmers Market Directory API, Google Fonts API.




Canopy

Canopy is a go-to sharing resource for web developers, programmers and engineers. My team and I set out to build a resource where live projects, code snippets, and interaction between developers are all in one place. Live projects are embedded and rendered on the site so that visitors can interact with them without leaving Canopy.

Technologies Used: Ruby on Rails, postgreSQL, AJAX, jQuery, Screenshot Machine API, highlight.js, Paperclip, Bcrypt, RSPEC.




Scope

Scope is a tool for people who want to learn to code, but don't know what resources to use or need help with the research process. Users can save resources that interest them, as well as comment on resources they've already used in order to help other users find the right tools. I created a clear, organized database of resources and tags that doesn't exist anywhere else.

Technologies Used: Ruby on Rails, postgreSQL, AJAX, jQuery, Bootstrap, Bcrypt.




Simplify Local

Top 6 Semi-Finalist at Mastercard Masters Of Code Hackathon

In 24 hours, my team and I built a mobile-friendly web app where users can find and shop at locally owned businesses near them. I lead the front-end development by designing layouts, integrating maps, and developing the pitch to the judges.

Technologies Used: Ruby on Rails, login and authentication with Twitter Digits API, mobile payments with the MasterCard Simplify Commerce API, HTML 5 location services, Mapbox.
